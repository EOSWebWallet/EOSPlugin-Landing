EOS Plugin

EOS Plugin is a bridge that give you a chance you to visit the future world of distributed web in your browser today. It allows you to run EOS dApps right in your browser without running a full EOS node.

Open source software: work with dApps, create anonymous identities, send/receive tokens, sign transactions, etc. You can install the EOS Plugin add-on in Chrome and Firefox.

Our mission is to make EOS as easy to use for as many people as possible.
